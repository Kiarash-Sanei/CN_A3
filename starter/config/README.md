# Runtime Configuration

این پوشه برای فایل‌های configuration دانشجو در زمان حل تمرین در نظر گرفته شده است.

در starter، هیچ table entry کامل، policy آماده یا route کامل قرار داده نشده است. دانشجو باید بر اساس طراحی pipeline خودش، entryهای لازم را بسازد و در گزارش کوتاه توضیح دهد.

نمونه‌هایی که می‌توانید خودتان اضافه کنید:

- فایل command برای `simple_switch_CLI`
- یادداشت mapping پورت‌ها به subnetها
- فایل‌های test ورودی

نام tableها و actionها در این تمرین عمدا از قبل تعیین نشده‌اند.
